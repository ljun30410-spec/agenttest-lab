# 联网搜索（Web Search Tool）测试用例

> 需求背景：当前 AI 问答暂无法联网搜索，导致内容实效性较差。本需求在 Leanin AI 问答的 Tool 中增加"联网搜索"，当用户的资料无法完整回答用户问题时，适时调用以达到更好的问答效果。
>
> 核心验收逻辑：
> 1. 资料可完整回答 → **不应**调用联网，聚焦资料内容；
> 2. 资料部分相关（缺失外部信息）→ **应**调用联网，联合"我们的方案/资料内容"与互联网信息一起回答；
> 3. 资料无相关内容（时效/外部话题）→ **应**调用联网，返回联网内容。

---
